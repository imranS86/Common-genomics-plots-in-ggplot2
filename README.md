# Common-genomics-plots-in-ggplot2
This repository presents the interactive report demonstrating how to generate commonly used genomics visualization plots (such as barplot, density plot, scatter plot, violin plot, and box plot) using the ggplot2 R package. The expression data is included in the repository to reproduce these plots. The pipeline is implemented in R using R Markdown, integrating the code and plots in an interactive report.
